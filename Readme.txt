Petunjuk Penggunaan Aplikasi :
(Pastikan untuk meng-install node.js versi terbaru dan vue.js)
(Pastikan sebelum menyalakan server aplikasi baik untuk tes-json atau tes-medigo lakukan npm install karena harus install dependency terlebih dahulu)
1. Buka command prompt/git bash dan pergi ke path folder tes-json berada(cd (path to folder tes-json)), 
Nyalakan server untuk data aplikasi dengan mengetik npm run test pada command prompt.
2. Setelah itu pergi ke path folder tes-medigo berada(cd (path to folder tes-medigo)), ketik npm install untuk
meng-install semua dependency yang digunakan untuk aplikasi ini, lalu setelah proses instalasi selesai nyalakan
server aplikasi dengan mengetik npm run serve.
3.Selamat anda berhasil menyalakan aplikasi, silahkan mencoba aplikasi yang telah dibuat
